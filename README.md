# Sales Performance Dashboard

This project is a **Power BI** dashboard designed to analyze and visualize sales, profit, and category performance.

## Features

* **KPI Tracking:** Real-time visibility of Total Sales and Total Profit.
* **Categorical Analysis:** Breakdown of sales performance by Category and Region.
* **Top Performers:** Identification of the highest-performing product categories.
* **Interactive Slicers:** Dynamic filtering by time, region, or category for deep-dive analysis.

## Tools Used

* **Power BI Desktop**
* **Microsoft Excel** (Data Source)

## Output

* **Power BI Dashboard:** The interactive `.pbix` file.
* **Exported PDF:** A static snapshot of the dashboard for reporting.
